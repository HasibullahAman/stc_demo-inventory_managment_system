# Shahir Trading Company

## Inventory Management System

A modern inventory management system developed for **Shahir Trading Company**, built to manage products, stock, and business data efficiently using a full-stack architecture.

This project is a demo system designed to showcase real-world usage of **Next.js 16**, **Prisma ORM**, and **MySQL**, deployed on **Render**.

---

## Features

- Product management (create, update, delete)
- Inventory and stock tracking
- Structured database schema with Prisma
- Server-side rendering and API routes with Next.js
- Scalable and production-ready architecture
- Ready for future authentication and role management

---

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **ORM:** Prisma
- **Database:** MySQL
- **Language:** TypeScript
- **Deployment:** Render

---
